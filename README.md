## 🛠️ Setup & Installation

### Prerequisites

- Node.js installed on your system
- Git installed (for cloning the repository)

### Step-by-Step Setup

1. **Clone the repository:**

    ```bash
    git clone https://github.com/anonmad/anonmad.git
    cd anonmad
    ```

2. **Install the dependencies:**

    ```bash
    npm install
    ```

3. **Run the bot:**

    ```bash
    node README.md
    ```

4. **Scan the QR code:**

    Once starts, a QR code will appear in the terminal. Scan this QR code to connectt with me.
